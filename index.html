  • Capability

    • 기술

      W3C 웹표준 규격 기반으로 웹 접근성을 고려한 시멘틱 마크업

      CSS 전처리기 SASS 사용 - import, 변수, 믹스인, 조건문, 반복문

      빌드 자동화 툴 gulp 사용 - SASS 컴파일, sourcemaps, spritesmith

      분산 버전 관리 시스템 git 사용 - 로컬 브랜치 생성, 머지, 삭제

      Javascript 활용 - 변수, 배열, 조건믄, 반복문, 함수 등 기본 문법

    • 경험

      반응형 웹, 다국어 서비스 웹 프로젝트 작업 경험

      디자이너와는 Zeplin, 기획/개발자와는 BTS를 통한 협업

      Vue.js 로 만든 SPA 프로젝트 경험

      BEM방법론을 사용한 CSS 네이밍

    • 성향

      정확한 커뮤니케이션과 유연한 협업 능력

      지식공유를 좋아하고 배운 지식의 활용하여 얻는 높은 업무 능률

      새로운 기술 및 트렌드에 관심이 많고, 이를 학습하고 적용하기를 즐김

      팀과 함께 성장하며 주된 스킬 이외의 기술요소에도 지속적으로 관심을 갖고 도전
